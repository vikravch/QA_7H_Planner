Task - одна задача:
 - id - идентификатор задачи - сгенерированное автоматически
 - deadline - дата и время до выполнения задачи - не обязательное поле
 - description - описание задачи - обязательное поле
 - status - выполнено\в процессе\запланировано\отложено - не обязательное поле
 	при создании новой задачи - статус запланировано по умолчанию
 - urgency - срочность (1..10) - обязательное поле
 - importance - важность (1..10) - обязательное поле
 
 Planner - инструмент управления задачами:
 - добавление задачи
 - печать списка задач
 - ввод задачи с консоли - дз
 - бекап задач в файл
 - считывание задач с файла
 - сортировка задач по срочности - дз
